# AAPL Stock Analysis 📊

## Objective

The goal of this project is to analyze Apple (AAPL) stock data using Python and understand key financial metrics such as returns, volatility, and overall performance.

---

## What This Project Does

* Collects real stock data using `yfinance`
* Calculates daily returns
* Measures risk (standard deviation)
* Visualizes price trends
* Analyzes cumulative returns over time

---

## Tools Used

* Python
* Pandas
* Matplotlib
* yfinance

---

## Key Results

* Average Daily Return: ~0.18%
* Risk (Volatility): ~1.25%

---

## Insights

* AAPL demonstrates relatively stable returns with moderate volatility.
* The cumulative return trend shows consistent long-term growth.
* Short-term fluctuations highlight the presence of market risk.
* This reinforces the importance of long-term investing and diversification.
