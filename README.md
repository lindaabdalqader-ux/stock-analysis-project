# AAPL Stock Analysis with Python

This project analyzes Apple (AAPL) stock using Python and real historical market data.

## Project Overview
The notebook downloads AAPL stock data and performs a basic financial analysis by:
- plotting the stock price over time
- calculating daily returns
- measuring volatility (risk)
- plotting cumulative returns to show investment growth

## Tools Used
- Python
- pandas
- matplotlib
- yfinance

## Key Metrics
The project calculates:
- **Average Daily Return**
- **Risk (Standard Deviation)**
- **Cumulative Returns**

## Files
- `aapl_stock_analysis (3).ipynb` — main notebook with the analysis

## Why This Project Matters
This project demonstrates beginner-level quantitative finance skills, including:
- working with real financial data
- return and risk analysis
- data visualization
- investment performance tracking

## Next Steps
Future improvements could include:
- comparing AAPL against other stocks
- adding moving average strategies
## Insights

- AAPL shows relatively stable daily returns with moderate volatility.
- The cumulative return trend indicates overall growth over the period analyzed.
- Short-term fluctuations highlight market risk, reinforcing the importance of diversification.
## Objective

The goal of this project is to analyze stock data using Python and understand returns, volatility, and overall performance of a stock.
## How to Run

1. Install dependencies:
   pip install yfinance pandas matplotlib

2. Run the notebook:
   jupyter notebook aapl_stock_analysis.ipynb
