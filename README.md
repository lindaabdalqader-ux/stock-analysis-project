# AAPL Stock Analysis with Python

This project analyzes Apple (AAPL) stock using Python and real historical market data.

## Project Overview
The notebook downloads AAPL stock data and performs a basic financial analysis by:
- plotting the stock price over time
- calculating daily returns
- measuring volatility (risk)
- plotting cumulative returns to show investment growth

## Tools Used
- Python
- pandas
- matplotlib
- yfinance

## Key Metrics
The project calculates:
- **Average Daily Return**
- **Risk (Standard Deviation)**
- **Cumulative Returns**

## Files
- `aapl_stock_analysis (3).ipynb` — main notebook with the analysis

## Why This Project Matters
This project demonstrates beginner-level quantitative finance skills, including:
- working with real financial data
- return and risk analysis
- data visualization
- investment performance tracking
